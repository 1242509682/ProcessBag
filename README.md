## ProcessBag 进度礼包
```
作者：少司命    
修改：羽学    
源码地址：https://github.com/Controllerdestiny/TShockPlugin/tree/master/ProgressBag  
这是一个Tshock服务器插件主要用于：服务器内的玩家击败指定Boss领取对应的奖励，可配置更多的物品ID内容与领取条件。   
```

## 插件版本
```
更新日志   
v1.0.7  
1.修复了世吞克脑同进度领取问题（用怪物图鉴进一步判断NPCID）  
  
v1.0.6  
1.加入更多的进度配置项  
2.更改了进度名【独眼巨鹿】为【鹿角怪】  
  
v1.0.5  
1.修复执行命令忽略权限  
2.优化配置文件读取  
  
v1.0.4  
1.适配1.4.4  
2.执行指令，无视权限执行(有些指令权限在方法内部判断无法无视执行)  

v1.0.3  
1. 可以组合进度限制，在配置文件中和对一个礼包进行多个进度限制。  
2. 新增限制领取组，在配置文件中修改，可添加多个组，可领取组保持为空则所有组都可以领取。  
3. 新增指令，该指令可一键领取所有满足领取条件的礼包。  
```
## 命令
```
/礼包 help  
/礼包 list --礼包列表  
/礼包 reload --重读配置  
/礼包 重置 --重置领取玩家  
/礼包 领取 礼包名 -- 领取指定礼包  
/礼包 领取全部  -- 领取已击败BOSS的礼包  
```
## 权限
```
bag.use --领取礼包  
bag.admin --管理礼包  
```
## 权限示例
```
/group addperm default bag.use  
```
